# TDAudioSwitch

TDAudioSwitch ist ein einfaches PowerShell-CLI-Tool fuer Windows zum schnellen Umschalten von Standard-Audiogeraeten.

Geplant ist folgende Funktionalitaet:

- Verfuegbare Ausgabegeraete auflisten
- Verfuegbare Mikrofone auflisten
- Per Tastatureingabe ein Standard-Ausgabegeraet waehlen
- Per Tastatureingabe ein Standard-Mikrofon waehlen
- Mit der Leertaste nichts aendern und das Tool direkt beenden

## Projektstruktur

- `main.ps1`: Hauptskript mit der CLI-Logik
- `TDAudioSwitch.bat`: Bequemer Starter fuer Windows

## Start

Vorerst sind die Dateien nur als Grundgeruest angelegt. Spaeter soll das Tool ueber die Batch-Datei oder direkt per PowerShell gestartet werden.

## Anforderungen

- Windows
- PowerShell
